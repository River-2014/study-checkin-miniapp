# 学习打卡小程序 - 长期记忆

## 项目概况
- **项目名**：小升初冲刺打卡微信小程序
- **目标用户**：11-12岁小升初学生及家长
- **技术栈**：微信小程序原生框架，本地存储(wx.setStorageSync)，无云服务依赖
- **核心功能**：每日学习任务打卡、专注计时、星星积分、奖励商城、成就徽章、错题本、AI出题练习、家长控制台

## 产品战略团队产出（2025-07-09）
- 已产出完整产品战略三件套，存放于 `deliverables/product-strategy/`
- PRD含27项需求（P0-10项/P1-10项/P2-7项），总估算111人天
- 北极星指标：DECR（每日有效打卡率），目标≥60%
- 核心定位：习惯养成×学科练习×亲子契约 三环闭环
- 蓝海位置确认，防御窗口约6个月
- 最大短板：社交功能缺失
- 最大壁垒：积分兑换真实家庭奖励+家长管控闭环
- 积分通胀风险：默认7天315分可换最贵奖励
